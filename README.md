# 🎬 Netflix Movies Analysis: Highest vs Lowest Grossing

## 📌 Project Overview
This project analyzes the **highest and lowest grossing movies of all time** to identify patterns in ratings, runtime, genres, and directors. The goal of this analysis is to understand the characteristics that are commonly associated with successful blockbuster movies compared to low-performing films.

The project follows a complete **data analytics workflow**, including data preparation, exploratory data analysis, and dashboard visualization.

Python was used for **data cleaning and analysis**, while **Power BI** was used to build a visual dashboard to present insights in a clear and interactive way.

---

## 📊 Dataset
Two datasets were used for this project:

1. **Top 1000 Highest Grossing Movies of All Time**
2. **15 Lowest Grossing Movies of All Time**

The datasets include information such as:
- Movie title
- IMDb rating
- Runtime
- Genres
- Release year
- Directors
- Number of votes

The datasets were combined and analyzed to compare trends between the two categories.

## 🧹 Data Preparation
The following steps were performed during data preparation:

- Loaded datasets using **Pandas**
- Added a **category column** to differentiate between highest and lowest grossing movies
- Checked and handled **missing values**
- Combined datasets for comparative analysis
- Extracted and analyzed **genre information**

---

## 📈 Data Analysis
The analysis focused on several key areas:

### Average IMDb Rating
Compared the average IMDb rating of highest and lowest grossing movies.

### Average Runtime
Analyzed differences in runtime between successful and low-performing movies.

### Genre Analysis
Identified the most common genres appearing in high and low grossing movies.

### Director Analysis
Explored directors who appear frequently among blockbuster movies.

---

## 📊 Dashboard (Power BI)
A **Power BI dashboard** was created to visualize the insights from the analysis.

The dashboard includes:
- Average IMDb Rating comparison
- Average Runtime comparison
- Genre distribution analysis
- Top directors in high-grossing movies

These visualizations help clearly highlight patterns between successful and unsuccessful movies.

---

## 🔍 Key Insights
Some important insights discovered during the analysis:

- Highest grossing movies generally have **higher IMDb ratings**.
- Blockbuster movies tend to have **longer runtimes**.
- **Action and Adventure** genres dominate high-grossing movies.
- Certain directors appear multiple times in blockbuster films.
